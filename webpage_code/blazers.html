Legend:

  • RK: Rank

  • EFG%: Effective Field Goal Percentage

  • PTS: Points Per Game
  • COACH-PLAYOFF: Number of playoff appearances achieved by teams under the leadership of the current main coach
  • NEW RK: New Rank

    • Why 'NEW RK'? Some player names are duplicated, and some plays have MP<10, so the original rank is not consistent. 'NEW RK' is corrected and is now consistent. Moreover, the original rank has ties, but because 'NEW RK' is used as one of the composite primary keys, it cannot include ties. If the original key has a tie, then the player with higher MP will have a higher 'NEW RK'.
